# project-6
Group 6 Project for DSCI445 @ CSU

## Bank Marketing Data

- Classification problem
- 16 variables, 40k+ observations
- *Target:* Has the client subscribed a term deposit?


Report will include data cleaning, model formulation, feature selection, model tuning, and finally model comparison and assesment. We hope to fit a variety of classification models (Logistic, LDA, QDA, KNN, Tree based methods, SVM) on this data set.
